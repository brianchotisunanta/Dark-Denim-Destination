# Dark Denim Destination

I designed a front-end application, called Dark Denim Destination, an online store that sells mens black denim.

## Web Technologies used:
* **_HTML5_**
* **_CSS3_**
* **_JavaScript_**
* **_Bootstrap_**
* **_AngularJS (Directives, Module, Controllers, Services)_**
* **_Angular UI Router_**
* **_Google Font Open Sans & Shadows Into Light_**
